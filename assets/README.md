# CIPP API Documentation Assets

This directory contains assets for the API documentation site.

## Required Assets

The following assets should be added to ensure complete branding:

### Favicons
- `favicon.ico` - Browser tab icon (32x32 or 16x16)
- `favicon-16x16.png` - 16x16 PNG favicon
- `favicon-32x32.png` - 32x32 PNG favicon  
- `apple-touch-icon.png` - 180x180 Apple touch icon

### Social Media
- `cipp-social.png` - 1200x630 Open Graph image for social sharing
  - Should display "CIPP API Documentation" text
  - Use CyberDrain brand colors: Navy (#003049) and Orange (#F77F00)

### Optional
- `cipp-logo.png` - Logo for documentation header (if needed)

## Creating Assets

You can generate these from the CIPP main application's existing branding or create new ones with:

**Colors:**
- Primary: Orange #F77F00
- Secondary: Navy #003049  
- Background: #F0F2F5 (light mode)
- Background: #0C121D (dark mode)

**Font:**
- Inter (same as main CIPP UI)
